# Wrapper to solomon for use by Ethan Pesikoff

Primary Goal: give a wrapper for Ethan Pesikoff (my collaborator in research) to use solomon easily and effectively.

Ethan Pesikoff is a math PhD student. He is proficient in JupyterLab and Mathematica, but not a programmer by training. He prefers simple interfaces, and doesn't want to use the terminal. He uses a Windows laptop.

Capabilities for the software:
- Define custom monodromies for arbitrary genus and number of punctures in terms of Dehn twists
- Has the option to name new curves on top of predefined curver curves, for custom use
- Calculate their orbit graphs
- Export the orbit graphs to png and live viewing of orbit graphs
- Export generators given the orbit graphs (see LefschetzDisk.ipynb) for use in GAP

Ideally this would exist in a programmable environment, so that Ethan can write basic code (e.g. for loops, while loops, etc.). JupyterLab would work well.

All the files for this project should live in their own directory, and Claude should suggest a few names for it :)
